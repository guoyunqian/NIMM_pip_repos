# 00temp/

存放算法整理过程中的**中间数据**（临时文件、对照样本、迁移备份等），**不纳入正式算法产出**。

## 约定

| 约定 | 说明 |
|------|------|
| 按算法分子目录 | `00temp/<算法代号>/`，如 `00temp/weather_phenom_grid_12h/` |
| 可删除 | 整理验证完成后中间文件可清理；重要对照请归档至 `test/` 或文档 |
| 勿提交大文件 | 格点样本、NC 等大体积数据勿入库；目录内已配置 `.gitignore` |
| 与运行输出区分 | 业务输出在 `PHENOM/`（`--output-dir`）；输入在外部 SCMOC；本目录仅服务**整理过程** |
