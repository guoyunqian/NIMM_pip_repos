# 00log/

原始算法整理过程的**日志**目录。

## 约定

- **一次整理过程对应一个日志文件**。
- 文件名建议：`<算法代号>_整理_<YYYYMMDD>.log`。
- 日志中须记录：
  - 整理起止时间与操作人；
  - 各步骤完成情况；
  - **仍有问题、需人工补充的目录或文件**（必填）；
  - 已迁入仓库的正式路径对照。
