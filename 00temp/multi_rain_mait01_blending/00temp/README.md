# 00temp — 算法整理中间数据目录

存放**原始算法整理过程**中产生的临时文件、对照样本、迁移备份等，**不纳入正式算法产出**。

## 使用约定

| 约定 | 说明 |
|------|------|
| 按算法分子目录 | 建议 `00temp/<算法代号>/`，如 `00temp/mait_1h/` |
| 可删除 | 整理验证完成后，中间文件可清理；重要对照结果请归档至 `test/` 或文档 |
| 勿提交大文件 | 格点/站点样本、HDF 等大体积数据勿入库；目录内已配置 `.gitignore` |
| 与运行日志区分 | 业务运行日志见 `log/`（由 `resource/mait_1.ini` 的 `log_file` 指定）；本目录仅服务**整理过程** |

## 当前子目录

| 子目录 | 用途 |
|--------|------|
| `mait_1h/` | MAIT 1h 整理过程中的中间样本、对照输出、迁移前备份等 |
