# 00temp/

存放算法整理过程中的**中间数据**（临时文件、对照样本、迁移备份等），**不纳入正式算法产出**。

## 约定

| 约定 | 说明 |
|------|------|
| 按算法分子目录 | `00temp/<算法代号>/`，如 `00temp/multi_optimize_tp_24h/` |
| 可删除 | 整理验证完成后中间文件可清理；重要对照请归档至 `test/` 或文档 |
| 勿提交大文件 | 格点/站点样本、HDF 等大体积数据勿入库；目录内已配置 `.gitignore` |
| 与运行数据区分 | 业务样例在 `resource/`；运行日志在 `log/`；本目录仅服务**整理过程** |

## 当前子目录

| 子目录 | 用途 | 状态 |
|--------|------|------|
| `multi_optimize_tp_24h/` | 本算法整理过程中间样本、对照输出等 | 仅 `.gitkeep` 占位；有文件时补充并记入 `00log/` |
