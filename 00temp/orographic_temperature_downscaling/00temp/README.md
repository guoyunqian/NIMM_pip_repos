# 00temp — 算法整理中间数据目录

存放**原始算法整理过程**中产生的临时文件、对照样本、迁移备份等，**不纳入正式算法产出**。

## 使用约定

| 约定 | 说明 |
|------|------|
| 按算法分子目录 | 建议 `00temp/<算法代号>/` |
| 可删除 | 整理验证完成后，中间文件可清理；重要对照结果请归档至 `test/` 或文档 |
| 勿提交大文件 | 格点/站点样本、HDF 等大体积数据勿入库；目录内已配置 `.gitignore` |
| 与仓库根目录区分 | 仓库根 `00temp/` 为算法中间目录集合；本目录仅服务**本算法整理过程** |

## 当前子目录

|--------|------|
