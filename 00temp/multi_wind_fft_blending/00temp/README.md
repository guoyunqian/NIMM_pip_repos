# 00temp — 算法整理中间数据目录

存放**原始算法整理过程**中产生的临时文件、对照样本、迁移备份等，**不纳入正式算法产出**。

## 使用约定

| 约定 | 说明 |
|------|------|
| 按算法分子目录 | 建议 `00temp/<算法代号>/`，如 `00temp/fft_merge/` |
| 可删除 | 整理验证完成后，中间文件可清理；重要对照结果请归档至 `test/` 或文档 |
| 勿提交大文件 | 格点样本、NetCDF 等大体积数据勿入库 |
| 与运行输出区分 | 样例输入在同级 `NIMM_pip_testdata/.../test_data`；本目录仅服务**整理过程** |

## 当前子目录

| 子目录 | 用途 |
|--------|------|
| `fft_merge/` | multi_wind_fft_blending 整理过程中间样本、对照输出、迁移备份等 |
