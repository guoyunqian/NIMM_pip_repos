# 00temp/

算法整理过程中的**中间数据**目录。

## 用途

- 存放从原始脚本/服务器拷贝的未整理代码、临时解压文件、对比结果等。
- 整理完成并迁入正式目录（`src/`、`utils/`、`cli/` 等）后，应清理或移出本目录对应子目录。

## 建议子目录命名

```
00temp/
  <算法名>_<YYYYMMDD>/     # 单次整理任务的中间文件
    raw/                   # 原始拷贝
    diff/                  # 对比 diff
    trial/                 # 试跑输出
```

## 注意

- 本目录内容**不纳入**正式算法代码，勿与 `src/`、`resource/` 混用。
- 大体积数据、密钥路径请勿提交版本库（见根目录 `.gitignore`）。
