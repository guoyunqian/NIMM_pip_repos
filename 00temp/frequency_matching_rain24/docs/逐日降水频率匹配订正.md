## 基本信息

- 算法名称：`frequency_matching_rain24`
- 原始工程名：`qpf_fm_rain24` / `optimize_tp_24h`
- 算法类型：`04single_calibration`
- 贡献人：郭云谦、陈荣、李东勇
- 原始路径：`D:\temp\202301_zhinengwangge\20230206_unitycode\NIMM_pip_repos\TEMP\260625\qpf_fm_rain24`

## 算法功能

该算法面向数值模式 24 小时累积降水预报，利用历史同期模式场与站点实况之间的统计关系，对当前预报进行相似个例检索、光流位移订正、切片频率匹配和全域格点频率匹配，输出订正后的 MICAPS3 站点产品和 MICAPS4/NetCDF 格点产品。

## 目录说明

- `src/`：核心算法源码，包含相似度计算、光流订正、频率匹配、插值、输出和主流程。
- `cli/`：命令行入口，支持 `python -m cli --plugin=...` 调用。
- `resource/`：运行配置、站点模板、掩码、模式 plugin JSON、原始运行日志和原始目录骨架。
- `test/`：工程结构、导入和 CLI help 测试。
- `nbs/`：算法说明 notebook 和配图。
- `docs/`：原始程序说明和本整理说明。
- `utils/`：运行配置读取、日志、多进程和数据准备工具。

## 主要入口

- 主流程：`src/correct_tp_24h.py::mainProcess`
- CLI：`cli/__main__.py`
- 配置读取：`utils/util_env.py`
- 主配置：`resource/optimize_tp_24.ini`
